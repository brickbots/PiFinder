Load a preset to see a shipped variant; green means the arrangement on screen reproduces the shipped table entry.

Fan-out checklist for a new variant

    
1. pointing_model/imu_dead_reckoning.py  - _q_imu2cam branch (from this tool)
2. pointing_model/imu_dead_reckoning_legacy.py - same branch in the legacy table
3. camera_interface.py                   - SCREEN_ROTATE_AMOUNTS entry
4. ui/menu_structure.py                  - PiFinder Type option list
5. main.py                               - display-rotate special case, if the
                                           display mounts rotated (cf. as_bloom)
6. tests/test_imu_dead_reckoning_equivalence.py - add key to SCREEN_DIRECTIONS
7. this tool                             - add the preset (with its board_rev)
   to the embedded JSON (tests/test_imu2cam_tool_presets.py enforces parity)
8. tests/test_imu_dead_reckoning.py      - paste the pytest block (optional)
9. docs/source                           - PiFinder Type user docs, if user-facing
